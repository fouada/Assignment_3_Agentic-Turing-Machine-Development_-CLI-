# Agent Skills

Claude Agent Skills for multi-agent translation pipeline.

## Skills

### 1. english-to-french-translator
Translates English → French, handles noisy inputs with spelling errors.

### 2. french-to-hebrew-translator
Translates French → Hebrew, bridges Romance and Semitic languages.

### 3. hebrew-to-english-translator
Translates Hebrew → English, completes round-trip translation.

### 4. translation-chain-coordinator
Orchestrates all three agents, manages pipeline execution.

## Usage

These skills are automatically loaded by `run_with_skills.py`.

To modify agent behavior, edit the SKILL.md file for the relevant agent.
